# Roadmap

## v0.1 scaffold and grid

- TypeScript, Vite, Phaser, and Vitest project scaffold.
- Core parser and level state separated from rendering.
- Tutorial room rendered as an 8x8 grid with placeholder-safe assets.

## v0.2 movement/pushing/undo

- Deterministic movement rules.
- Crate pushing.
- Undo and reset support.

## v0.3 doors/plates/win condition

- Pressure plate and door logic.
- Exit and room completion detection.
- Basic level progression.

## v0.4 enemies started

- Sleepy golem parser, state, rendering, and tutorial levels.
- Deterministic post-player golem movement.
- Caught state, undo/restart restoration, and preview markers.
- Golems can hold red pressure plates.

## v0.5 level editor

- In-browser room editor.
- Level validation and export.
- Fast iteration workflow for new rooms.

## v0.6 daily seeded puzzle

- Seeded room generation.
- Daily puzzle selection.
- Shareable seed and completion metadata.
