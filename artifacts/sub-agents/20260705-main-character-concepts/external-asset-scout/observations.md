# External Asset Scout - Main Character Search

Status: complete

## What It Saw

- Free rigged/animated character options exist, but none matched the requested distinctive playful lead mascot without substantial customization.
- Best practical free sources were KayKit Adventurers plus KayKit Character Animations, and Quaternius Ultimate/Base Characters plus Universal Animation Library.
- Mixamo has broad animation coverage, but raw asset redistribution is not a good fit for this public web-game repo.

## Decisions

- Use free CC0 packs as fallback/retargeting references, not as the final mascot identity.
- Proceed with a generated Meshy mascot candidate because a distinctive original main character is justified.

## Caught Issues

- Direct orange fox/platform mascot visuals risk looking too close to established characters.
- Any generated model still needs license/provenance review and runtime optimization before gameplay integration.
